# 如何在本地运行项目

1. 前提：nodejs
2. 进入文件夹，输入npm install

   ```
   npm install
   ```
   
4. 等待安装依赖
5. 输入以下指令，启动项目
   ```
   npm start
   ```
   
7. 之后会自动访问`localhost:3000`
